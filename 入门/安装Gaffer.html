Gaffer 中文文档

安装Gaffer

Gaffer软件包是一个自包含的文件，因此您需要手动安装它，并在必要时手动配置它。一旦提取之后Gaffer目录包含完整的应用程序可供使用。

Note

为了与Linux/macOS最佳实践保持一致，我们将演示如何在/opt/目录中安装Gaffer。我们还使用Windows上的C:\software\作为示例安装路径。当然您也可以将它安装到系统上您具有写入和执行访问权限的任何位置。

在linux上安装

在linux上安装Gaffer:

  1. 下载Gaffer
  2. 打开终端
  3. 提取文件: 
    user@desktop ~ $ cd ~/Downloads
user@desktop ~/Downloads $ sudo tar -xzf gaffer-1.2.2.0-linux.tar.gz -C /opt/

Gaffer现在就安装到了/opt/gaffer-1.2.2.0-linux

在Windows上安装

在Windows上安装Gaffer:

  1. 下载Gaffer
  2. 提取文件(建议使用7-Zip)
  3. 将解压出来的gaffer-1.2.2.0-windows文件夹移动到你想安装的位置，比如C:\software

Gaffer现在安装到了C:\software\gaffer-1.2.2.0-windows

Note

Gaffer需要在Windows上安装Microsoft Visual C++ Redistributable这需要在启动Gaffer之前下载并安装。
